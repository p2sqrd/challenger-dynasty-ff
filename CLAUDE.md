# Working preferences (Pranav)

- When a change is complete, verified, and pushed, **open a PR for it** as
  part of finishing the work — don't ask whether to open one. New request =
  new PR, reset onto the latest `main` first.
- Default to suggesting the **simplest / cheapest** option first, but lay out
  all viable options from the outset rather than one at a time.
